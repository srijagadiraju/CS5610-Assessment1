Useful HTML Tags

  1. div tags are useful for grouping elements
  2. nav tags help represent a navigation bar/menu
  3. p tags are useful for adding paragraphs
  4. h1-h6 tags are useful for adding headers (largest to smallest)
  5. a tags allow you to add links to other websites/documents
  6. ul tags allow you to add unordered lists (bulletted)
  7. ol tags allow you to add ordered lists (numbered -- what I used to create this!)
  8. li tags define a new element to add in a list
  9. section tags allow you to organize and separate code
  10. footer tags allow you to define the footer for a page

HTML Tags for SEO

SEO stands for Search Engine Optimization.
This describes the process of having pages "rank" higher when looked for using search engines, which can lead to more users on the website.

Most Important Header Tags

  1. title tags allow us to create headlines that can draw readers to our website and should have important keywords
  2. meta tags allow us to describe the information on the page
  3. h1-h6 tags should have the more important content for search engines to know what the page is about
  4. alt text for images makes the website more accessible
  5. span tags can provide additional information about the page, like ratings, publishing date, etc.

Font Styling/Sizing

You may want to use 12/14/16/20px fonts when you want control over the look and size of your elements.
For example, this could be for your nav bar, subheadings/captions, paragraphs, or for important content that requires a larger heading.
Absolute units (ex: px) can be used when you want precise control over content, while relative units (ex: em, rem) allow for flexibility/responsivity and are designed based on the sizes of other headers.

  • Item 1 - 12px
  • Item 2 - 14px
  • Item 3 - 16px
  • Item 4 - 20px
  • Item 5 - 1em (relative to the font size of the parent)
  • Item 6 - 2rem (relative to the font size of the root element)

Positioning Options

Four Options:

  • Static Positioning: Default positioning.
  • Relative/Absolute Positioning: Positioned relative to its normal position. Can stack them.
  • Fixed Positioning: Follows as you go down the page.
  • Sticky Positioning: Will follow once you get to it and will continue with you for the remainder of the page. Positioned based on the user's scroll position.
